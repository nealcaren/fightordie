Interactive Embedding Plot in Quarto (CSV Data)

Interactive Embedding Plot

This section demonstrates an interactive plot of document embeddings, with data loaded from a CSV file.

Code 
Plot = require("https://cdn.plot.ly/plotly-latest.min.js")

// Load and process the CSV data
data = FileAttachment("embeddings.csv").csv({ typed: true })

// Calculate the range for both x and y to make the plot square
xRange = d3.extent(data, d => d.x)
yRange = d3.extent(data, d => d.y)
range = d3.extent([...xRange, ...yRange])

// Create the plot
chart = {
  const trace = {
    x: data.map(d => d.x),
    y: data.map(d => d.y),
    mode: 'markers',
    type: 'scatter',
    text: data.map(d => d.title),
        font: {family: "Roboto Serif, serif"},
    hoverinfo: 'text',
    marker: {
      size: 8,
      color: 'rgba(128, 0, 32, 0.3)',  // Constant color with some transparency
      line: {
        color: 'rgba(31, 119, 180, 1)',
        width: 0
      }
    },
    customdata: data.map(d => d.url)
  };

  const layout = {
    font: {family: "Roboto Serif, serif"},
    plot_bgcolor: 'rgb(250,250,250)',
    paper_bgcolor: 'rgb(250,250,250)',
    hovermode: 'closest',
    xaxis: {
      showgrid: false,
      zeroline: false,
      showticklabels: false,
       autorange: true
    },
    yaxis: {
      showgrid: false,
      zeroline: false,
      showticklabels: false,
      autorange: true
    },
    width: 600,
    height: 600,
    margin: {l: 10, r: 10, t: 10, b: 10}
  };

  const plot = Plot.newPlot('myPlot', [trace], layout);
  
  // Add click event listener
  document.getElementById('myPlot').on('plotly_click', function(plotData) {
    const point = plotData.points[0];
    const url = point.customdata;
    if (url) {
      window.open(url, '_blank');
    }
  });

  return plot;
}

The plot above shows document embeddings loaded from a CSV file. Hover over points to see document titles, and click on points to open the associated URLs.

Explanation

This interactive plot visualizes document embeddings in a 2D space. Each point represents a document, with its position determined by the embedding coordinates.

  • Hover: Moving your cursor over a point will display the title of the document.
  • Click: Clicking on a point will open the associated URL in a new tab.

This visualization can help in understanding the relationships between documents in the embedding space, potentially revealing clusters or patterns in your data.
